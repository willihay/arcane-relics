## [1.6.0]

### Changed

Updates to Wand Enchanting Table:
- emit light and appear lit only when table contains lapis
- render input wand on top of table
- keep contents when broken

## [1.5.0]

Initial public release of **Arcane Relics** for **Minecraft 1.21.11** on **Fabric**.

### Added

- the **Arcane Wand** as the base wand item
- the **Wand Enchanting Table** for enchanting and recharging wands
- seven specialty wands:
  - Fang Wand
  - Fireball Wand
  - Ice Wand
  - Levitation Wand
  - Lightning Wand
  - Regeneration Wand
  - Wind Wand
- charge-based casting and recharge mechanics for specialty wands
- alternate recharge sources tied to specific mobs or blocks for most wands
- advancement progression for enchanting and recharging wands
- optional integrations for **JEI**, **Mod Menu**, and **YACL**
- localized language files for multiple non-English languages
- world-local server configuration for gameplay balance and Wand Enchanting Table behavior
