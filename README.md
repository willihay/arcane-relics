# Arcane Relics
_A collection of powerful relics from crafty wizards_

**Arcane Relics** is a Minecraft mod that adds arcane wizardry to Minecraft. It includes the following:
- Powerful wands that capture the power of elemental magic and other forms of energies found in the Minecraft world.
- A **Wand Enchanting Table** to turn the base **Arcane Wand** into one of the more powerful versions, as well as recharge depleted wands. 

## Installation
**Arcane Relics** is a [Fabric](https://fabricmc.net/) mod. You must have the Fabric Loader installed to use this mod.

To install **Arcane Relics**, follow these steps:
1. Download the latest version of the mod from the [releases page](https://github.com/willihay/arcane-relics/releases).
2. Place the downloaded `.jar` file into your Minecraft mods folder.
3. Launch Minecraft and enjoy the arcane magic!

## Usage
At a high level, you first need to craft an **Arcane Wand** and a **Wand Enchanting Table**. The base **Arcane Wand** by itself is inert and useless.
Use the **Wand Enchanting Table** to turn the base **Arcane Wand** into one of the more powerful versions using various items found in Minecraft, along with Lapis Lazuli as a catalyst.
Each wand has a limited number of charges. You can repeat the enchanting process with an enchanted wand and the same ingredient to recharge the wand. 
You can even transform one wand into another by enchanting with a different item. 
Most wands also have another way to recharge the wand for free!

For more information, please refer to the [Arcane Relics Wiki](https://github.com/willihay/arcane-relics/wiki).
